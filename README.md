## Features
- Feedback form with name and message
- Serverless backend to handle POST requests
- Real-time response after submission
- Clean and minimal UI

## Tech Stack
- HTML, CSS, JavaScript
- Vercel Serverless Functions

## How It Works
- The frontend sends feedback using a POST request
- A Vercel serverless function processes the request
- A success response is returned to the user
